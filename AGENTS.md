# AGENTS.md

## Project

InterviewPrep Laravel Application

A personal Laravel application for organizing technical interview preparation.

---

## Stack

* Laravel 13
* Blade
* Tailwind CSS
* MySQL
* Laravel Breeze
* Vite
* Groq API
* Laravel Http facade

---

## Rules

* Always explain the plan before coding
* Work feature by feature
* Never build the whole app at once
* Use resource controllers
* Use Eloquent relationships
* Use validation
* Use clean architecture
* Use Blade only
* Use Tailwind only
* Do not install unnecessary packages

---

## Workflow

1. Explain feature architecture
2. Create migration
3. Create model
4. Create relationships
5. Create controller
6. Create routes
7. Create views
8. Test feature

---

## AI API Rules

* Use Http facade only
* API keys in .env only
* Save AI results in database
* Handle API errors gracefully

---

## Important

Before generating code:

* explain files
* explain logic
